# Idea_feedback_module
The Idea Feedback Module is a web-based application designed to collect ideas from users and allow others to provide feedback on those ideas. The system enables users to submit ideas with a title and description, store them securely, and attach multiple feedback entries to each idea. This project helps in organizing ideas efficiently and encourages constructive feedback in a simple and user-friendly manner. The application is implemented using HTML, CSS, and JavaScript, with data stored using browser localStorage.Submit ideas with proper details View all submitted ideas Add multiple feedback entries for each idea Store data persistently without using a backend server To design a simple and interactive idea submission system To allow multiple feedback entries for each idea To store ideas and feedback persistently.Thechnologies are used such as HTML5 – For creating the structure of web pages,CSS3 – For styling and layout ,JavaScript – For application logic and interactivity,localStorage – For client-side data persistence.
